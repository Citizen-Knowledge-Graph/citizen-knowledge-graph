Scenarios


There are two types of scenarios you can run:
  • "What if" scenarios: change some data points to see how results of inquiries might change or what is then missing to fill something out.
    Note: Inquiry patterns can be defined to get activated specifically for changing a data point from one value to another. This is useful to get more information when running scenarios or to get timely change-related information when something has actually changed.
  • "How to get there" scenarios: pick scenarios or describe a status you want to reach and see which data point would need to change in order to reach this goal.
🔗 You can connect your graph to someone else's graph, your partners graph for instance. Then you can run federated inquiries together.
